# Episode: tenex-edge positioning doctrine: citizenship demoted, proof-as-hero, Nostr hidden

## Prior State

README led with 'Citizenship for your agents' as the hero tagline. CLI about string read 'identity + awareness on the Nostr fabric.' No competitive positioning framework, no explicit vocabulary guardrails, and no structural separation between shipped reality and aspirational vision in outward-facing copy.

## Trigger

Four-agent research swarm + Fable creative-direction pass produced convergent findings: (1) 'citizenship/sovereign identity' vocabulary is adjacent to crypto/network-state discourse — a verified brand liability for the target dev audience; (2) 'Nostr' as a keyword is colonized by crypto-speculation projects (Clawstr, Agentry) while crypto dev activity is collapsing ~75%; (3) 'orchestration' and 'swarm' are saturated across ~90+ competitor tools and signal 'yet another dashboard'; (4) the cross-host @mention terminal transcript is the one demo no competitor in the entire category can screenshot; (5) Claude Code's native Agent Teams is the single biggest structural threat — copy must explicitly address cross-host and cross-session/cross-machine durability or read as 'a worse free Anthropic feature'; (6) the product-spec docs themselves flag collision-coordination, multi-app society, and cross-person features as aspirational/unbuilt, and the beachhead persona as an open question.

## Decision

Committed positioning doctrine: category label is 'an identity and awareness fabric for coding agents' (never 'orchestrator,' 'agent OS,' or 'swarm'). Hero = a real @mention terminal transcript (proof-as-headline, T4) carrying the pain hook in the subhead and identity durability as beat two. 'Citizenship for your agents' demoted from hero to one earned use in the architecture/philosophy section. 'Nostr' banned above the architecture section — described by properties ('self-owned keys, no central server to trust, works if a relay dies') until revealed once in 'How it works.' Banned vocabulary list established: orchestrate, swarm, autonomous, conductor, sovereign, decentralized, web3, crypto, seamless, supercharge. Present-tense reserved as a certification of shipped-and-tested status; everything else gets explicit 'we haven't built' or lives under a labeled horizon heading. Mandatory 'What this isn't yet' section as the anti-hype signature move.

## Consequences

- README structure locked to 7 beats: proof-transcript hero → pain in audience's verbatim words → shipped-floor bullet list → identity-as-foundation with Agent Teams/hcom/mcp_agent_mail comparison table → how-it-works with Nostr reveal + corruption war story as trust asset → 'what this isn't yet' honest section → quickstart per host + FAQ
- Every present-tense claim in copy must be verified as shipped-and-tested before writing; the docs-vision agent's grounded/hype flag list becomes the certification source
- The signature line 'the host is a body; the identity is the person' is reserved for the identity-foundation beat, not the hero
- Agent Teams comparison is placed at beat 4 (correct altitude), not in the hero — converts the structural threat into a differentiator without defining the product relative to a competitor
- CLI about string and other in-product text that currently names Nostr prominently may need alignment with the new vocabulary doctrine
- Creative tension between collision-prevention hook (what devs click on) and identity-durability moat (what's ownable) resolved: hook leads, proof carries, moat follows — doctrine's own warning that coordination is 'a hypothesis, not a pillar' is preserved in the 'what this isn't yet' section

## Open Tail

- Actual README.md rewrite and landing-page copy not yet executed in this session — only the creative-direction brief is committed
- Nostr naming policy applies to outward-facing copy but CLI about string still reads 'on the Nostr fabric' (line 865) — may need update for consistency
- Beachhead persona remains an officially open question per product-spec; copy must not assert a confident 'built for X' the docs haven't settled
