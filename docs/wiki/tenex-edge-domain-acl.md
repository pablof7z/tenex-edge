# Tenex-Edge Domain ACL

## Domain ACL Structure

The ACL feature remains intentionally removed; any stale ACL references are treated as refactor debris to delete rather than restore. (Previously: The domain had two verb planes with ACL (is_member) as a predicate both planes consulted, and an injection hook surfaced unauthorized agents to the human for decision.)
