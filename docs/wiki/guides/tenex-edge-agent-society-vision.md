# tenex-edge Agent Society Vision

## The Inside-Out Operating System

The higher-level abstraction revealed by the todo-app/podcast-app example is that the operating system is being turned inside out: apps become citizens with roles in a society, the human dissolves as the integration layer (no longer the manual courier between siloed apps), and agents self-organize around the user's goals with the human as a privileged node/oracle rather than the operator. <!-- [^8a3eb-3] -->

tenex-edge reframed is the citizenship protocol for an agent society spanning every app in a user's life — not just a coordination tool for a dev fleet. The dev-collision problem is one narrow slice of dissolving the human-as-glue across all software. <!-- [^8a3eb-4] -->
