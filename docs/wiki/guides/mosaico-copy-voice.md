# Mosaico Copy Voice

## Banned Vocabulary

The following words are banned in all copy: orchestrate/orchestration, swarm, autonomous, conductor, agent OS, sovereign, decentralized, web3, crypto, seamless, supercharge, teammates-that-never-sleep, and revolution/future-of. None of these appear in any published sentence.

<!-- citations: [^75f62-c8916] [^75f62-fe39f] -->
## Name Specific Host Tools

Copy names the specific host tools — Claude Code, Codex, OpenCode, Grok — rather than generic placeholders like 'AI agents' or 'AI teammates.' If a sentence would say 'your AI agent,' rewrite it to name the tool. <!-- [^75f62-92844] -->

## Present Tense Means Shipped

Present tense in copy certifies that a feature is shipped and tested. Unshipped features use explicit hedging language like 'we haven't built' or appear under a horizon heading. Never describe something in present tense that does not yet exist. <!-- [^75f62-84e8c] -->

## Verbs Over Adjectives

Copy leads with verbs — broadcasts, injects, persists, survives, lands. Adjectives are deleted wherever a sentence still reads without them. If an adjective-free sentence survives the cut, the adjective was not load-bearing. <!-- [^75f62-a0037] -->

## Numbers as Engineering Texture

Numbers appear in copy as engineering texture, not marketing boasts. 'One daemon per machine, one SQLite store, one relay connection' is on-voice. '38.5k LOC of blazing-fast Rust' is off-voice. Use quantities to convey architecture, not to impress. <!-- [^75f62-f2d2d] -->

## Register and Punctuation

Copy uses no exclamation points, no rhetorical hype-questions, and no emoji headers. The only permitted register shift is dry wit. <!-- [^75f62-9c68d] -->

## Category Label

The category label for mosaico in all outward copy is 'an identity and awareness fabric for coding agents,' never 'orchestrator,' 'agent OS,' or 'swarm.' <!-- [^75f62-6bd96] -->

## Governing Metaphor

The governing metaphor for mosaico copy is 'the fabric' — a thin, always-on layer floating above every terminal where agents exist as persistent beings while hosts remain disposable bodies. <!-- [^75f62-6ab29] -->

## Nostr and Mechanism Words

The words 'Nostr,' 'crypto,' and 'decentralized' never appear above the architecture section in outward copy; above the fold the mechanism is described by its properties — self-owned keys on disk, no central server to trust, works if a relay dies. The word 'Nostr' appears exactly once in outward copy, in the 'How it works' section, framed as an open protocol that is self-hostable with no vendor lock-in. <!-- [^75f62-1cec0] -->

## Active Voice

The @mention feature description uses active voice: 'Address a session and Mosaico delivers a real turn,' not the passive 'a message is delivered.' <!-- [^75f62-7914e] -->

## No Social Proof or Urgency Devices

Social proof (logos, stars, testimonials) and urgency/scarcity devices are deliberately absent from the copy because they would violate the honest-over-sensational voice and backfire with the hype-allergic developer audience. <!-- [^75f62-3aafd] -->
