# TENEX Overview

## Overview

TENEX is a multi-agent AI coordination system for software development built on the Nostr protocol, treating AI context as the primary artifact, with agent-to-agent communication flowing as cryptographically signed Nostr events. <!-- [^8a3eb-32] -->
