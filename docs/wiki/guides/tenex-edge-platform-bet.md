# tenex-edge Platform Bet

## Platform Bet

The platform bet for tenex-edge is a thin open adapter on Nostr where the protocol is the product; any host emits and consumes signed events, and tenex-edge owns no agents and no server. tenex-edge has no concept of any specific host (pc, Claude Code, etc.); it exposes a generic host-agnostic boundary.
