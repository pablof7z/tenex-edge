# tenex-edge Cross-Person Collaboration

## Cross-Person Collaboration

Tenex Edge enables cross-person collaboration where one user's agent can query another person's agent about how they handled something in one of their projects. Cross-person request/response payloads use NIP-44 encryption, and when metadata must also be hidden, NIP-59 gift wrap (kind:1059) is used. <!-- [^8a3eb-10] -->
