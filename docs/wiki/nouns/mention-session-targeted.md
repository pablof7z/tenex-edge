# mention (session-targeted)

A p-tagged Nostr kind:9 event addressed to another agent's pubkey, server-side-routed into that specific session's inbox, and if the session has a live PTY session, injected as a literal conversational turn — host-neutral delivery at the terminal level, not via a host API.
