# aggregate REQs

Three stable broad Nostr subscriptions replacing the old per-(project×kind) narrow-filter model: #h (all channels), #p (all durable ordinal pubkeys), and group-state by #d. Each aggregate backfills a large stored-event backlog down the relay connection.
