# unknown-task

A fallback workflow used when no existing workflow applies, meaning: refine the ask only as much as it deserves, do simple/reversible tasks directly, ask questions only when the answer materially changes priority/authority/access/money/reputation/irreversible state, capture an initial workflow once the task shape is clear, execute, then review.
