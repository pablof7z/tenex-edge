# ordinal label

An ordinal-qualified agent-instance label: agent_ordinal_label(slug, ordinal) = format!("{slug}{ordinal}"), e.g. @developer3. Ordinals are recycled per room occupancy, not durable — the label is a leasing artifact, not a permanent identity. This model was deleted in the redesign.
