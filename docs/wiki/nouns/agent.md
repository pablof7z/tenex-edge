# agent

A role that a session plays on the fabric, discovered from the supported harness's
native agent definitions or explicitly bound under
`<mosaico_home>/agents/<slug>.json`. The addressable, running unit is the session:
it derives its own key, publishes presence, and coordinates in channels. Agents
self-organize — the value is shared awareness that lets the left hand know what
the right hand is doing, not a durable per-agent identity.
