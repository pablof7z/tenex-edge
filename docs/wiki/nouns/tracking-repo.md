# tracking repo

The user-facing tracking repository the chief of staff shares with the user — the same repo that holds projects/, decisions/, and the daily report; the chief-of-staff home directory contents are migrated into it (mirroring the local path as <repo>/.agents/homes/chief-of-staff/) and symlinked back so state is shared across machines.
