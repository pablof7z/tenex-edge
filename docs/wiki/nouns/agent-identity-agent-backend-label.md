# agent identity (agent@backend-label)

The durable AGENT-INSTANCE = (slug, backend-label) → agent@backend-label (e.g. codex@laptop, haiku1@myBackend). @ always means backend label, never project. Backend labels are config.json backendName values, preserved exactly after trimming — not DNS hostnames, not slugified.
