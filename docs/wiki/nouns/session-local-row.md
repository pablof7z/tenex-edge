# Session (local row)

A local runtime keyed by its authoritative pubkey. It carries a private
monotonic runtime generation, agent role slug, channel, harness, process and
transcript metadata, liveness, timestamps, title, and activity. Harness resume
tokens and endpoints live separately as typed locators.
