# backend flag (kind:0 is_backend)

A boolean extracted from a kind:0 profile tag, parsed in fabric/provider/profiles.rs and persisted into relay_profiles.is_backend; used to identify daemon/mgmt keys so their traffic can be filtered out of agent-visible snapshots and rosters.
