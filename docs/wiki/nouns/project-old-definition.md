# project (old definition)

A top-level NIP-29 group owning a workspace (git checkout + machine); the only distinction from a channel is whether parent_hint is set. Later collapsed entirely — project becomes an attribute (a workspace binding) a channel may carry, not a separate concept, table, or code path.
